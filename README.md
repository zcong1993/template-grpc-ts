# template-grpc-ts

> grpc template for ts

## Usage

Install [SAO](https://github.com/saojs/sao) first.

```bash
yarn global add sao
# or
npm i -g sao
```

### From npm

```bash
sao template-grpc-ts my-project
```

### From git

```bash
sao zcong1993/template-grpc-ts my-project
```
